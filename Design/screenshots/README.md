# Screenshot Gallery Placeholders

This folder contains placeholders and descriptions for screenshots to be captured and uploaded for the repository documentation.

## Required Screenshot Views

Capture these views using a high-resolution macOS display under dark-mode settings:

1. **dashboard_mockup.png**: The main Cinematic Workspace view showing the drop zone, selected metadata, active pipeline switches, and the preset estimator pane.
2. **queue_active.png**: The Processing Queue showing active job rows, dynamic progress bars, and expanded console logs.
3. **dependency_manager.png**: The Dependency Manager showing scan statuses, tool paths, and Homebrew commands for missing libraries.
4. **privacy_screen.png**: The Privacy page displaying local directory paths and local security assurances.
5. **preferences_grouped.png**: The Settings view showing default configuration paths and default overwrite choices.

## Formatting Guidelines

- Dimensions: Capture standard macOS window sizes (e.g. 1920x1280).
- Color Mode: Utilize dark-mode system preferences.
- Visual Quality: Do not overlay device frames (laptops, monitors) unless explicitly required. Crop directly to the window border.
