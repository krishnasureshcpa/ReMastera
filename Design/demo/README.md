# Video Demo and GIF Guidelines

This directory contains placeholders for promotional and instructional video animations demonstrating ReMastera in action.

## Required Animations

1. **render_progress.gif**: A compressed loop showing a video file dragged onto the drop zone, the preset configured, and the job successfully sent to the queue.
2. **queue_execution.gif**: A loop showing the active encoding progress bars advancing, log lines appending in real time, and the "Reveal in Finder" action.

## Technical Specifications

- Format: Use high-quality WebP or compressed GIF format.
- Frame Rate: Maintain a steady 15 to 30 frames per second.
- File Size: Keep under 10 MB for fast loading on GitHub.
- Capture Range: Keep captured area limited to the ReMastera window to maintain focus.
